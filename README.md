- install wasm target system

- [install npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)
- install npx package:
  `npm install -g npx`
- clone the repo:
  `git clone https://github.com/Jeremy-Gstein/seemsgood_guild.git`
